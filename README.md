# Armario Virtual - Unity

Aplicación en desarrollo realizada con Unity para la gestión de un armario virtual.

## Tecnologías
- Unity
- C#
- Interfaz 2D

## Funcionalidades actuales
- Añadir prendas al armario
- Clasificación por tipo
- Visualización de prendas

## Estado del proyecto
Proyecto en fase de desarrollo como práctica de aprendizaje.
